MLB 2024 Season Analytics: Performance Insights and Statistical Analysis

Project Overview

This comprehensive analysis explores key performance metrics from the 2024 MLB season, focusing on home run race dynamics, pitching effectiveness, and comparative team analysis. Using advanced statistical modeling and data visualization techniques, we uncover patterns and insights that drive strategic decision-making in modern baseball analytics.

Analysis Framework

  • Data Source: Official MLB statistics via pybaseball API (Full 2024 season)
  • Visualization Tools: Interactive Plotly dashboards and animated bar chart races
  • Statistical Methods: Performance tracking, trend analysis, and comparative metrics
  • Focus Areas: Home run production, pitching analysis, and team performance comparisons

Key Insights

  • Home Run Leaders: Dynamic race throughout 2024 season with key momentum shifts
  • Pitching Performance: ERA and strikeout rate correlations reveal optimal performance profiles
  • Team Dynamics: Yankees vs Dodgers comparison highlights contrasting strategic approaches
  • Seasonal Patterns: Performance trends reveal critical periods affecting playoff positioning
